# Skill Exchange Hub

> A community platform connecting learners, mentors, and collaborators through learning contracts, roadmaps, office hours, and skill-based rooms.

**Features**
- **Learning:** Create and follow learning roadmaps and contracts.
- **Mentorship:** Mentor dashboards, batch management, and office hours.
- **Community:** War rooms, code arena, and profile management.
- **Admin:** User and mentor management, moderation, and analytics.

**Tech stack**
- **Frontend:** React (Create React App)

**Prerequisites**
- Node.js (>=14) and npm installed

**Quick Start (development)**
```powershell
cd "C:\Users\Admin\Desktop\skill-exchange-hub"
npm install
npm start
```

**Build**
```powershell
npm run build
```

**Tests**
```powershell
npm test
```

**Project structure (important files)**
- **Root:** [package.json](package.json) — project metadata and scripts
- **Public:** [public/index.html](public/index.html)
- **Entry:** [src/index.js](src/index.js)
- **App:** [src/App.js](src/App.js)
- **Navbar:** [src/components/Navbar.js](src/components/Navbar.js)
- **Pages:** [src/pages](src/pages)

**Deployment**
- The app builds to the `build/` folder via `npm run build`. Host the static files on GitHub Pages, Netlify, Vercel, or any static host.
